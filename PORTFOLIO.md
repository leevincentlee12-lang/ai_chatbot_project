# Adaptive Algebra Learning Platform Portfolio Notes

Use this file as a portfolio or application checklist. It is not a replacement
for the README; it is a shorter presentation guide for screenshots and talking
points.

## Project Summary

Adaptive Algebra Learning Platform is a Flask-based educational platform for
guided algebra practice, structured explanations, answer checking, lessons, and
progress tracking.

Public demo: https://ai-chatbot-project-tlou.onrender.com

## Suggested Demo Flow

1. Open the homepage and explain the project purpose.
2. Ask the system to solve a linear equation.
3. Show the response sections: answer, method, why, check, and next step.
4. Open Practice Mode and complete one generated problem.
5. Open the Progress page and show that activity is being tracked.
6. Open About and explain the architecture flow.
