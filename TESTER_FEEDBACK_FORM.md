## Suggested Test Tasks

Ask each tester to try at least three tasks so the feedback is based on real
usage:

- Solve an equation in Step-by-Step Mode, for example `solve 3x^2 - 4x - 4 = 0`
- Ask for a hint, for example `give me a hint for 2x + 4 = 10`
- Try Practice Mode and check an answer
- Submit one incorrect answer and review the feedback
