Hard Practice Upgrade

The old hard mode mostly changed numbers. The upgraded hard mode changes the algebra structure.

Before

Same Linear Pattern

Hard Linear Equation
9x + (-3) = 42
  • Still followed one familiar form: ax + b = c.
  • Difficulty mostly came from larger numbers.
  • Students could learn the pattern instead of the algebra.
After

Varied Algebra Structures

Variables On Both Sides
-10x + 13 = -11x + 11
Factorable Quadratic
x^2 - 5x + 6 = 0
  • Hard mode now includes variables on both sides, brackets, and quadratics.
  • The answer checker accepts complete solution sets like x = 2 or x = 3.
  • Incorrect answers get targeted feedback, including missing-root mistakes.

This comparison summarises the practice generator and checker upgrade made from student feedback.
