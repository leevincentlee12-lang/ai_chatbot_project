# Screenshot Folder

Save final project screenshots here using these names:

- `homepage.png`
- `solving-equations.png`
- `practice-mode.png`
- `lessons-page.png`
- `dashboard.png`
- `deployment.png`
- `architecture.png`
- `application_screenshots/07_test_results.png` for automated test evidence

Keep screenshots consistent: same browser width, clean sample inputs, no
personal information, no debug logs, and no empty loading states.

For before/after evidence, use the PNG files in
`improvement_before_after_images/`.
