System Overview

What Askademic can do right now

Askademic is a lightweight tutoring app built on Flask with a rule-based subject router and a symbolic maths helper. Its strongest area is algebra: it can solve equations, compare algebraic expressions, validate recorded steps, and generate practice questions with answer checking.

Current strengths

  • Multi-subject question routing for maths, science, English, and humanities
  • Symbolic equation parsing with support for user-style input such as `x^2`
  • Practice problems with answer checking and mastery tracking
  • Lesson endpoints, feedback logging, and analytics summaries

Math workflow

Question -> subject classifier -> maths engine
        -> solver or validation path
        -> structured explanation
        -> optional practice and answer checking

Example prompts

  • solve 3x + 5 = 20
  • factor x^2 - 9
  • what caused WW1
  • give me a TEEL paragraph

Design choices

The app now uses one consistent backend contract instead of mixing in-memory state with missing JSON files. The frontend practice flow is also aligned with the backend, so generated problems, submitted answers, and progress views now refer to the same state model.
