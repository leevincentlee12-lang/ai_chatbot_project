# Improvement Screenshot Captions

Use these captions if the application asks for evidence of improvements made
after testing or feedback.

## 01_general_question_handling_before_after.png

**Improvement shown:** The platform moved from a restrictive response to a more
helpful learning-focused response. Instead of only saying the system is
restricted, it explains what the algebra platform can do and redirects the user
toward supported learning tasks.

## 02_harder_practice_before_after.png

**Improvement shown:** Practice questions became more challenging. Earlier hard
practice examples were too simple, so the platform was improved to include more
advanced algebra such as non-monic quadratics, algebraic fractions, and rational
equations.

## 03_working_space_before_after.png

**Improvement shown:** A working-space feature was added so students can write
their steps before checking an answer. This supports mathematical reasoning
rather than only asking for a final answer.

## 04_common_mistake_feedback_after.png

**Improvement shown:** The platform gives feedback when a student submits an
incomplete or incorrect answer. This is evidence of progress toward the teacher
feedback about supporting common mistakes. The system has since been extended
with additional rule-based feedback for selected coordinate geometry and graph
feature mistakes.

## 05_progress_tracking_after.png

**Improvement shown:** The platform records learning activity and displays
progress indicators such as accuracy, difficulty, skill scores, and recent
activity.

## Recommended Application Set

If you can only include three improvement screenshots, use:

1. `02_harder_practice_before_after.png`
